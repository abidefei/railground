RailGround Formal Model

The RailGround Formal Model constitutes the foundation for the Interlocking Formal Model. It defines basic concepts and fundamental entities as well as modelling principles upon which the complex model can be created.

The formal model is structured in layers, each layer extending and refining the previous ones:

  • 1. Rails - Rail entities and rail topology
  • 2. Element Position - Movable elements position
  • 3. Paths - Paths and Topology traversal
  • 4. Path Lifecycle - Path lifecycle
  • 5. Vacancy Detection - Track Vacancy Detection
  • 6. Occupation Impact - Track Occupation impact on Paths and Signals
  • 7. Signals - Associated Entities and Signals
  • 8. Signal Dependency - Signal Dependency on Path
  • 0. Visualization - Model Animation and Visualization
